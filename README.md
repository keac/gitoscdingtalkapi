# Gitoscdingtalkapi
Git@osc dingtalk api git@osc 使用钉钉机器人提示api接口

（表情包功能暂未开放）

需要设置两个地方 index.php  

设置请求密码
$pwd = "Northme"; //Set password

设置钉钉机器人access\_token可在钉钉机器人详情页获取
$access_token = "xxx"; // Set Api token
